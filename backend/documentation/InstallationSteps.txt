Install FastAPI and Uvicorn using:
pip install -r requirements.txt

To run the server, use the line:
uvicorn main:app --reload
(If the above line doesn't work, instead try the line below:)
python -m uvicorn main:app --reload

Interactive API documentation can be viewed at the /docs url of the server, for example:
http://127.0.0.1:8000/docs
